Floricultura

Este projeto foi um estudo de caso para criar uma identidade visual completa para uma floricultura fictícia. Foi desenvovido o manual da marca, incluindo logotipo, paleta de cores, tipografias, grafismos e aplicações em diversos materiais, como cartões de visita, sacolas e mockups de site e aplicativo.
