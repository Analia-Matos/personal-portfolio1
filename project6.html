Case de estudo - TripIdeias

Processo de desenvolvimento

Este projeto é um case de estudo envolvendo todo o contexto de ux para uma empresa fictícia chamada TripIdeias. A Tripideias é uma agência de viagens que vende pacotes turísticos personalizados para oferecer uma experiência única para seus clientes. Foi desenvolvido todo o projeto desde o planejamento de pesquisa, pesquisa de mercado, definição de personas, objetivos, arquitetura de informações, e protótipos de média e alta fidelidade.
